Basics

Select a scenario from the list in the top-right of the page.
Press Ctrl-Enter in the editor to run the scenario with a new version of your code.
The game calls your tick() function 60 times per second.

Ship Control

  • api.position() → vec2: Get the current position in m/s.
  • api.velocity() → vec2: Get the current velocity in m/s.
  • api.heading() → float: Get the current heading in radians.
  • api.angular_velocity() → float: Get the current angular velocity in radians/s.
  • api.accelerate(acceleration: vec2): Linear acceleration. X axis is forward/back, Y axis is left/right.
  • api.torque(acceleration: float): Angular acceleration. Unit is radians/s2.
  • api.fire_weapon(): Fire the ship's main weapon.
  • api.explode(): Self-destruct.

Vector Math

  • vec2(x, y) → vec2: Create a vector.
  • v1 +- v2 → vec2: Basic arithmetic between vectors.
  • v */ float → vec2: Basic arithmetic between vectors and scalars.
  • -v → vec2: Negate a vector.
  • v.magnitude() → float: Magnitude (length).
  • v.normalize() → float: Normalize to a unit vector.
  • v1.dot(v2) → float: Dot product.
  • v1.distance(v2) → float: Distance between two points.
  • v1.angle(v2) → float: Angle between two vectors.
